视觉信息抽取（Visual Information Extraction, VIE）旨在挖掘、分析和提取视觉富文档中包含的关键信息字段。例如，给定一张收据的图像，VIE算法将告诉诸如商店名称、产品详情、价格等信息。对于表格等文档，VIE算法将抽取出其包含的键值对。
